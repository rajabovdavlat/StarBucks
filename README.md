# Starbucks Landing Page (In Progress)

A single-page landing page inspired by Starbucks, built from scratch using **pure HTML** and **CSS**.

**This project is still under active development.**

## Current Status

- Currently working only with desktop layout
- Mobile & tablet responsiveness (fully adaptive design) — **in progress**
- JavaScript interactivity — **planned for the next stage**

## Goal of the project

This is a **personal practice project** created to improve front-end skills, especially:

- Semantic HTML5
- Modern CSS (Flexbox, Grid, responsive design)
- Clean and structured code
- Pixel-perfect layout
- Mobile-first approach (coming soon)

## Features (planned)

- Fully responsive design (mobile, tablet, desktop)
- Smooth hover effects and transitions
- Interactive elements (menu, buttons, etc.) with JavaScript
- Clean and maintainable code structure

## Technologies (so far)

- HTML5
- CSS3 (Flexbox, modern properties)

JavaScript will be added later.
